**Mental Preparation System – Pre-Trade Focus Feature**

We want to add a Mental Preparation system designed to help traders pause before trading and improve discipline. This should feel like a quick optional focus ritual, not a long process.

This should be accessible from the main menu and optionally triggered before logging a trade.

---

**Mental Preparation Tab**

Add a new section in the menu called:

**Mental Preparation**

Purpose:
Help users mentally reset before trading.

This should contain customizable preparation tools.

---

**Mental Preparation Content Options**

Users should be able to toggle what they want to see:

Show Trading Quote

Show General Quote (movies, books, anime, philosophy, etc.)

Show Personal Affirmation

Show Breathing Exercise

Show Religious Reading (optional)

Users can enable or disable any of these.

Example toggles:

☑ Trading Quote
☑ General Quote
☐ Religious Reading
☑ Breathing Exercise
☑ Personal Affirmation

System should only show enabled items.

---

**Trading Quotes**

These should be discipline-focused quotes about:

Patience

Risk management

Emotional control

Consistency

Example:

"Amateurs trade emotionally. Professionals trade their plan."

Quotes should rotate randomly.

---

**General Quotes**

These can include quotes from:

Movies

Books

Anime

Philosophy

Sports

Focus on themes like:

Discipline

Patience

Perseverance

Self control

Example:

"Hard work beats talent when talent doesn't work hard."

Quotes should rotate randomly just like trading quotes.

---

**Personal Affirmations**

User can add their own affirmations.

Example:

"I only trade my edge."

"I protect my capital."

"I stay disciplined."

User should be able to:

Add affirmation

Edit affirmation

Delete affirmation

App should randomly show one.

---

**Breathing Exercise**

Add a simple breathing animation:

Example:

Inhale 4 seconds

Hold 2 seconds

Exhale 4 seconds

Total time should be short (10–15 seconds).

Add skip option.

Goal:
Quick emotional reset.

---

**Religious Reading (Optional Feature)**

This should be optional and hidden unless enabled.

User can select religion:

Example:

Islam

Christianity

Judaism

Hinduism

Buddhism

Other (future expansion)

App should display:

Short discipline-related verses from that religion.

Example themes:

Patience

Self control

Wisdom

Discipline

Respectful neutral presentation only.

No commentary.

Just text.

---

**Pre-Trade Mental Preparation Option**

Add optional toggle in settings:

**Require Mental Preparation Before Trade Logging**

ON / OFF

If ON:

When user presses:

Log Trade

Mental Preparation screen appears first.

After completion:

Button:

**I'm Ready To Trade**

Then journal opens.

If OFF:

User can still access Mental Preparation manually from menu.

---

**Skip Tracking (Future AI Use)**

Track:

Times user completes preparation

Times user skips preparation

This data may be used later by AI to detect behavior patterns.

Example future insight:

"You perform better on days you complete mental preparation."

---

**Design Goal**

Mental Preparation should feel:

Fast

Clean

Optional

Psychology focused

This should feel like:

"A quick discipline reset before trading."

Not:

A long motivation section.
